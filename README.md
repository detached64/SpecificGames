# SpecificGames

This repo contains tools for specific games.

## Games

[Humanity](https://vndb.org/v4209)

[Kochuuten](https://vndb.org/v19312)

[Renaissance](https://vndb.org/v611)

[Koukan Jouken](https://vndb.org/v13723)

[Youjo Ranbu~ Yogosareta Seiiki~](https://vndb.org/v5523) & [Youjo Ranbu 2](https://vndb.org/v5543)

## Build

1. Install [.NET Framework 4.8 Developer Pack](https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48).
2. Run `msbuild SpecificGames.sln /p:Configuration=Release`.
